---
title: "[New] Creating a new user"
---

Starting in April 2026, we are migrating all SIM Management user to our new Role-Based User Management System.

This article is related to the new system. If your account’s users have not yet been migrated, please refer to the articlers concerning the [old user management system](managing-editing-user-accounts.md)

To create a new user, you need to be an Admin yourself, to have access to the “Manage Users” menu.

![image](/knowledge-center/assets/image-1.png)

Fill in the mandatory information

![image](/knowledge-center/assets/image-2.png)

Then in the Privileges section below, click on “Add a new role”

![image](/knowledge-center/assets/image-3.png)

You can now define the first role you want to add to this user, and to which entity it should apply.

![image](/knowledge-center/assets/image-4.png)

You will see that certain roles have “Group” version and an **Restricted** “Account” version:

![image](/knowledge-center/assets/image-5.png)

If you choose Account, the role will be applied to the whole account.

If you select the Restricted “Group” version, you will have the possibility to select groups in the account to which this role will apply – instead of the whole account.

Once you have selected a role, the next menu will display the Business Entities that can be selected

This is screenshot, I can select between 3 Accounts:

![image](/knowledge-center/assets/image-6.png)

You can select the whole account:

![image](/knowledge-center/assets/image-7.png)

Or, if you had selected a Group version of a role, you can use the little down arrow to display the Groups, and limit your user’s role to specific Groups in this account

![image](/knowledge-center/assets/image-2.png)

Once you have finished this first role, you can now add new ones

Either to expand the permissions – for example Finance Manager and Customer Care on the same business entities, or to give a different role on a different business.

Here is an example where the user will have 3 different roles on 3 different entities:

![image](/knowledge-center/assets/image-2.png)

Once you have finished adding roles, click on “create user”

![image](/knowledge-center/assets/image-8.png)

An email will be sent to the new user, inviting them to finalize their account by choosing a password.
