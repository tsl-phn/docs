**Managing user accounts**

To be able to manage user accounts in the interface, your own user account must have the rights to read and manage users: “Any\_User\_Write” and “Any\_User\_read”  
Please note that you cannot modify your own user account.  
  
To manage users, click on the Admin icon and select Manage Users  
  
![image](/knowledge-center/assets/image-1.png)
  
For each user, you have the following actions available:  
  
![image](/knowledge-center/assets/image-2.gif)
  
– Details: shows the user’s settings (read only view)  
– Clone: enables you to duplicate this user account to create a new user. [Click here to learn how](https://docs.transatel.com/user-management/cloning-an-existing-user-account/)  
– Activate / Deactivate : enables you to deactivate the user account, or reactivate a deactivated account. ([click here for further info](https://docs.transatel.com/user-management/deactivating-or-deleting-user-accounts/))  
– Edit: gives you access to the edit menu, where you can change details (name, email adress) as well as edit the rights & scopes of the user  
– Delete: enables you to delete the user ([click here for further info on deletion](https://docs.transatel.com/user-management/deactivating-or-deleting-user-accounts/))
