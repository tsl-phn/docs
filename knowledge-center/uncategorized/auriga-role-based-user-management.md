# Auriga Role-based User Management

**What is Role-based user management?**

RBUM is a smart way to control access in your Auriga platform. Instead of managing permissions for each user individually, you assign **roles** that define what actions they can perform. This makes administration easier, faster, and more secure.

**Why Roles Matter?**

**Roles** group permissions into easy-to-understand sets.  
**Permissions** define what users can do (e.g., view SIM details, manage bundles).  
Users can have **multiple roles**, combining permissions for flexibility.

**Standard Roles Available**

We’ve designed ready-to-use roles to cover most needs:  
**Read Only** – View SIM details, usage, and reports. No changes allowed.  
**Customer Care** – Support subscribers with individual actions like activation or suspension.  
**Fleet Manager** – Manage entire fleets, including bulk actions and automation rules.  
**Finance Manager (Read Only)** – Access invoices and billing reports without making changes.  
**Developer (Read Only)** – API integration and technical tools for IT teams.  
**Admin** – Full access to all features, including user management and advanced settings.

**Permissions Explained**

**Features**: Groups of permissions for easy role creation.  
**Scopes**: Detailed permissions for advanced control (usually not needed for most customers).

✔ Simplifies user management  
✔ Improves security and compliance
