Transatel's Developer Platform gives you programmatic control over your mobile subscribers, SIM cards, and network services. Whether you're managing a fleet of IoT devices or running a virtual mobile network, the platform's REST APIs let you activate subscribers, send SMS, monitor network usage, handle number portability, and manage your product catalog — all from a single authenticated interface.

## How It Works

<Steps>
  <Step title="Obtain credentials">
    Your Transatel account manager provides a `client_id` and `client_secret`. Keep these secure — they never appear in requests directly.
  </Step>
  <Step title="Get an access token">
    Exchange your credentials for a short-lived Bearer token using the Authentication API. Tokens expire in 3600 seconds and must be refreshed.
  </Step>
  <Step title="Call the APIs">
    Include the Bearer token in the `Authorization` header of every request. APIs are grouped by domain: Connectivity Management, Network, and OCS Catalog.
  </Step>
  <Step title="Handle events">
    Configure a webhook endpoint in your system to receive real-time events whenever subscribers change state, portability completes, or SMS delivery reports arrive.
  </Step>
</Steps>

## API Domains

| Domain | Base URL | What it does |
|--------|----------|--------------|
| Authentication | `https://api.transatel.com/authentication` | Obtain OAuth 2.0 access tokens |
| Connectivity Management | `https://api.transatel.com/connectivity-management/subscribers` | Manage subscribers, SIM cards, and portability |
| Network Attach | `https://api.transatel.com/network/attach` | Location history and network detach |
| Network SMS | `https://api.transatel.com/network/sms` | Send and retrieve SMS messages |
| Network OTA | `https://api.transatel.com/network/ota` | Over-the-air SIM network switching |
| Network Data Session | `https://api.transatel.com/network/data-session` | Active data session diagnostics |
| Network Usage | `https://api.transatel.com/network/usage` | CDRs and usage aggregates |
| OCS Catalog | `https://api.transatel.com/ocs/catalog` | Product catalog and product factory |

<Note>
All API calls require a valid Bearer token in the `Authorization` header. See [Authentication](/authentication) to get your first token.
</Note>
