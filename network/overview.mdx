The Transatel Network APIs give you real-time and historical visibility into how your SIM cards interact with mobile networks worldwide. Whether you need to investigate a connectivity problem, dispatch an SMS, push a SIM onto a private network over the air, or pull detailed usage records for billing and analytics, every operation is available as a straightforward REST call authenticated with your Bearer token.

All Network API requests require an `Authorization: Bearer <token>` header. Individual endpoints are guarded by fine-grained OAuth 2.0 scopes, so you can grant each integration only the permissions it needs.

## Authentication

Every request must include a valid Bearer token obtained from the Transatel authorization server. Pass it in the `Authorization` header:

```http
Authorization: Bearer <your_access_token>
```

Each API section documents the specific OAuth 2.0 scopes required for its endpoints. Request only the scopes your integration needs.
